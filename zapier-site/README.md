## Customization

### Colors
The main color scheme can be modified in `styles.css`:
```css
:root {
    --primary-color: #ff6a00;
    --primary-hover: #e55a00;
    --text-dark: #1a1a1a;
    --text-light: #666;
    --bg-light: #fafafa;
    --bg-dark: #1a1a1a;
}
```

### Content
- Update text content in `index.html`
- Modify company logos in the integrations section
- Change statistics and numbers as needed

### Animations
- Adjust animation timing in `styles.css`
- Modify JavaScript animation parameters in `script.js`

## Performance Optimizations

- **CSS**: Optimized animations using transform and opacity
- **JavaScript**: Efficient event listeners and intersection observers
- **Images**: Placeholder images for logos (replace with actual assets)
- **Fonts**: Google Fonts with display=swap for better loading
